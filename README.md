## ✨ Features
- **Modern UI/UX**: Designed with a focus on glassmorphism and premium aesthetics using Tailwind CSS 4.0.
- **Smooth Animations**: Powered by Framer Motion for seamless transitions and interactive elements.
- **Interactive Sound Effects**: Integrated `use-sound` for an immersive browsing experience.
- **GitHub Integration**: Live GitHub contribution calendar visualization.
- **Dynamic Contact Form**: Fully functional contact form with Nodemailer integration.
- **Responsive Design**: Optimized for all screen sizes, from mobile to ultra-wide displays.
- **State Management**: Robust and lightweight state handling with Zustand.

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm / yarn / pnpm

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/mohdmusaiyab/new-portfolio.git
   cd new-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add your credentials:
   ```env
   # Example variables
   EMAIL_USER=your-email@gmail.com
   EMAIL_PASS=your-app-password
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
